<h3>BIM Content packages</h3>
<p>ARCHICAD introduces a flexible way to classify elements according to any national or company standard classification system and makes available ARCHICAD Properties that can be connected to any construction element or zone, based on any Element Classification.</p>
<div>
<p>ARCHICAD enables the one-step import of BIM data that have been predefined in an external ARCHICAD file (PLN, PLA or TPL format). With the         <!-- --><a href="https://helpcenter.graphisoft.com/?p=65757" target="_blank">Import BIM Content function</a>, you can import the following BIM Content to your ARCHICAD project:</p>
<ul>
    <li>A Classification System</li>
    <li>Any Properties belonging to that Classification System</li>
    <li>The IFC Translator “Mapping” presets based on that Classification System:
    <ul>
        <li>Type Mapping for IFC Import</li>
        <li>Type Mapping for IFC Export</li>
        <li>Property Mapping for IFC Export</li>
    </ul>
    </li>
</ul>
<p>If the BIM Content package below contains only the Classification System itself we provide it in XML format. These XML files can be imported in the <a href="https://helpcenter.graphisoft.com/?p=65301" target="_blank">Classification Manager</a>.</p>
</div>
